-- Dictionary 

A dictionary stores data in key : value pairs.
syntax: dict_name={ key : value, }

    Creating a Dictionary
        student = {
            "name": "Rahul",
            "age": 20,
            "course": "BCA"
        }

        print(student)

Output: {'name': 'Rahul', 'age': 20, 'course': 'BCA'}
