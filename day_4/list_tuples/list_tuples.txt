-- list 
A list is used to store multiple values in a single variable.
    list are mutable
    have index
    it can store values of any type
eg: Marks = [12,34,54,24,54,56]
    names = [12,34,54,"hello","hi"]

-- tuples -- 
A tuples is used to store multiple values in a single variable.
    list are immutable
    have index
eg: Marks = (12,34,54,24,54,56)
