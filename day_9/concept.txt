--- python provides several methods to handle file 

- To open file = open('file name','mode')
- it takes two arguments first is "file name" and second is "mode" in which that file shoulld be opened  
- The mode can be : 
      1. 'r' for reading                 
      2. 'w' for writting                       
      3. 'a' for appending
      4. 'x' for create mode .it create the file and if file is already exist it shows error.
      5. 't' it specifis that how we want to open the file . it specifies the text file . it is by default
      6. 'b' it specifies to open the file as binary



# 1. to append a file
'''
    f=open('for_file_io.txt','a')
    f.write(" hello world")            #for append the file. here write is used.
    f.close
'''

#   to read ,write or append combine
    #here we use -- r+ ; w+ ; a+
    # r+ -- no trunkated . read and write.
'''
    f=open('for_file_io.txt','r+')
    f.write("hello here") # it over write the things in the other till the words we have written. printer goes to the starting.
    print(f.read()) # it will read the things after we have over written. 
    f.close 
'''

# w+ -- this is reading and writing . in tis case file gets trunkated i.e complete wiped out. 
'''
    f=open('for_file_io.txt','w+')
    f.write("hello world present here")
    f.close 
'''

# a+ -- read and append . This is the stream is positioned at the end of the file. file is not trunkated.
'''
    f=open('for_file_io.txt','a+')
    f.write("hello world present here")
    f.close 
'''

# *note--we have to close the file by f.close() , so to avoide it we use with open('file name','r') as f: to automaticallu close file
